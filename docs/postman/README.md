# Evidências de Testes e Coleção Postman

Este diretório armazena a coleção de requisições (em formato `.json`) exportada do Postman ou Insomnia, bem como prints de tela evidenciando a execução dos casos de teste.

- `collection.json`: Exportação da coleção de requisições (inclua este arquivo aqui).
- `prints/`: Pasta opcional para armazenar imagens PNG/JPG com evidências (ex.: `CT01_login.png`, `CT02_criar_paciente.png`).

Se você ainda não possui a coleção ou prints de testes, marque os itens como [PENDENTE] no relatório e inclua-os aqui assim que estiverem disponíveis.
